COLORS In HTML

Colors in HTML work on a hexadecimal system of 0-9 and A-F while the common decimal system we use 0-9. It uses these values in a hash code in the format #000000 where each 2 numbers corresponds to red, green and blue.

Examples

The yellow hash code is #FFFF00
The magenta hash code is #FF00FF
The aqua hash code is #00FFFF
The white hash code is #FFFFFF
The red hash code is #FF0000
The green hash code is #00FF00
The blue hash code is #0000FF
The black hash code is #000000
